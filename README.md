# 🏥 Vigilância Epidemiológica com IA

> Sistema de vigilância epidemiológica que detecta surtos precocemente cruzando dados de notificação, busca na web e clima, antecipando ações de saúde pública.

**🌐 Demo ao vivo (web app interativo):** https://jhondados.github.io/saude-epidemia-vigilancia/

[![Setor](https://img.shields.io/badge/Setor-Saúde-blue)]() [![IA](https://img.shields.io/badge/IA-ML-green)]() [![Demo](https://img.shields.io/badge/Demo-Interativa-orange)]() [![Dados](https://img.shields.io/badge/Dados-CSV%2FExcel-success)]()

## 📊 Resultados-chave (KPIs)
**21 dias** Antecip. · **detectados** Surtos · **nacional** Cobertura · **88%** Acerto

## 🧠 Tecnologias de IA
ML · Anomaly Detection · Epidemiology

## ✨ O que faz
- Detecção precoce de surto
- Previsão de casos por região
- Cruzamento multi-fonte
- Alerta à vigilância
- Apoio à decisão pública

## 🗄️ Fonte de dados real
[DataSUS/SINAN + clima](https://datasus.saude.gov.br) — bases de entrada e resultados disponíveis para download em CSV/Excel no app.
